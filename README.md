
# Bright Kids Autism - Aplikasi Edukasi Anak

Aplikasi edukasi interaktif untuk anak-anak (khususnya untuk membantu kemandirian dan literasi anak autis) menggunakan Next.js, Tailwind CSS, dan Web Speech API.

## Cara Menjalankan di Komputer Anda (VS Code)

1.  **Unduh Proyek**: Klik tombol **Download** (biasanya ikon tanda panah ke bawah) di panel atas atau samping Firebase Studio.
2.  **Ekstrak File**: Jika file berbentuk `.zip`, ekstrak folder tersebut.
3.  **Buka di VS Code**: Buka VS Code, pilih `File > Open Folder...`, dan pilih folder proyek yang sudah diekstrak.
4.  **Install Dependencies**: Buka terminal di VS Code (tekan `` Ctrl + ` ``) dan ketik:
    ```bash
    npm install
    ```
5.  **Jalankan Aplikasi**:
    Di terminal VS Code, ketik:
    ```bash
    npm run dev
    ```
6.  **Buka Browser**: Akses `http://localhost:9002`.

## Fitur Utama
- **Visual Menarik**: Desain penuh warna dengan tema ceria yang disukai anak-anak.
- **Suara Interaktif**: Menggunakan Web Speech API gratis untuk membacakan teks setiap aktivitas.
- **Navigasi Mudah**: Tombol besar dan ikon yang intuitif untuk mendukung perkembangan anak.

## Teknologi
- **Next.js 15**
- **Tailwind CSS**
- **Lucide React Icons**
- **Firebase Firestore (Data Storage)**
